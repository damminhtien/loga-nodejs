# Loga restful api v1.0.0
## Technology
* NodeJS
* Express.js
* Mongodb and mongoose
* ES6lint
* Nodemon
## Note
* Form tag must have enctype='multipart/form-data' to post with bodyPaser.
* GET: get all, find
* POST: create
* PUT: update
* DELETE: delete
* HTTP code: 200 - success, 500 - error in server, 300 - request redirect.
